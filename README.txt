                    PANOPTIQUE v1.0
              News Aggregator pour Windows
═══════════════════════════════════════════════════════════════

Merci d'avoir choisi PANOPTIQUE !

INSTALLATION RAPIDE :
───────────────────
1. Double-cliquez sur "📱 INSTALLER PANOPTIQUE.bat"
2. Suivez les instructions à l'écran
3. L'application s'ouvrira automatiquement

UTILISATION :
───────────
• Bureau : Double-cliquez sur l'icône PANOPTIQUE
• Menu Démarrer : Recherchez PANOPTIQUE

INSTALLER COMME APPLICATION :
─────────────────────────────
Après le premier lancement :
1. Cliquez sur l'icône d'installation dans Chrome (barre d'adresse)
2. Ou allez dans Paramètres → "Installer l'application"
3. L'app s'ouvrira alors comme une vraie application Windows

CONFIGURATION REQUISE :
──────────────────────
• Windows 10/11
• 4 GB RAM minimum
• 500 MB d'espace disque
• Connexion Internet pour la synchronisation

FONCTIONNALITÉS :
───────────────
✓ Agrégation de flux RSS
✓ Extraction d'articles web
✓ Mode hors ligne
✓ Synchronisation automatique
✓ Interface moderne
✓ Mode sombre
✓ Notifications Windows

DÉPANNAGE :
──────────
• Port 3000/3001 occupé : Fermez les autres applications web
• Erreur Node.js : Réinstallez depuis nodejs.org
• Page blanche : Videz le cache du navigateur (Ctrl+F5)

DÉSINSTALLATION :
───────────────
• Panneau de configuration → Programmes → PANOPTIQUE
• Ou exécutez DESINSTALLER.bat dans le dossier d'installation
